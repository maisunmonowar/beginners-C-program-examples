Are are assuming you are using VsCode. 

On Linux:
=========
Make sure you have compilers install. 
sudo apt install build-essential
gcc --version

You can compile manually and run by:
gcc HelloWorld.c
./HelloWorld

Or you can use the built-in GDB in the VsCode.
Install "C/C++ Compile Run" by danielpinto8zz6

Once installed you should see a play button appear on top of your code.
